## GooseV3Laptop GUI

This project currently contains a very simple GUI window as a starting point.

- **Language**: Python 3
- **GUI toolkit**: Tkinter (bundled with the standard Python installation)

### How to run

Make sure you have Python 3 installed and dependencies installed:

```bash
pip install -r requirements.txt
python main.py
```

This will open a resizable window with buttons that talk to the Jetson API (start/stop tracking).
